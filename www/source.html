Source Code

The moztraybiff source code can be accessed in either of the following ways:

  • Download a sources tarball for a certain release:
    • Version 1.0.4
      Versions:
      Mozilla 1.7, Thunderbird 0.7, Mozilla 1.7+ nightlies.
      Tarball:
      mozTrayBiff-1.0.4.tar.gz
      Notes:
      To compile for nightlies, run:
      make -DMOZ_TRUNK
  • View the source using the CVS web interface
  • Check out the source using anonymous CVS with the following commands (password is guest): 
    cvs -d :pserver:guest@mozdev.org:/cvs login
cvs -z3 -d :pserver:guest@mozdev.org:/cvs co moztraybiff

CVS clients and Help Documentation can be found at cvshome.org.
